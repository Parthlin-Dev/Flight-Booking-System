# Flight Booking System

## Project Overview

A Spring Boot based Flight Booking Management System developed to manage airports, flights, passengers, and bookings.

## Technologies Used
* Java
* Spring Boot
* Spring Data JPA
* MySQL
* REST API
* Postman

## Features
* Add Airport
* View Airport Details
* Add Flight
* Manage Passengers
* Book Flights
* Update Booking Status
* Delete Records

## Tools Used
* Eclipse
* MySQL Workbench
* Postman
* GitHub
